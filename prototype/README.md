# Prototype / Design Assets

This folder contains the original design prototypes and reference assets used before building the production Next.js app in `alphalead-ai/`.

## Contents

- **JSX prototypes** — early interactive mocks of screens and flows (`home.jsx`, `chat.jsx`, `ritual.jsx`, `night.jsx`, etc.).
- **Shared UI primitives** — `companion.jsx` (Alpha orb, icons, buttons), `browser-window.jsx`, `ios-frame.jsx`, `web.jsx`.
- **Tweaks panel** — `tweaks-panel.jsx` + `tokens.css` for runtime palette, typography, motion, and density tuning.
- **HTML preview** — `AlphaLead AI.html` static landing reference.
- **shots/** — exported screenshots and flow diagrams for documentation and pitch decks.

These files are kept for reference. The production implementation lives in the repository root (`../`).
